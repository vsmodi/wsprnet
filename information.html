Weak Signal Propagation Reporter Network

Information about WSPR

This Master’s graduate project explores the innovative adaptation of the Weak Signal Propagation Reporter (WSPR) protocol for real-time tracking of high-altitude balloons. Traditionally used for monitoring stationary transmitters, WSPR's static nature limited its application for dynamic objects. By integrating GPS data and leveraging technologies like HTML, CSS, JavaScript, and Leaflet.js, the project created an interactive mapping solution to visualize balloon trajectories in real time. This new system successfully addresses challenges in accuracy and usability, providing a more dynamic and insightful representation of moving transmitters.

Leaflet.js was specifically chosen for its lightweight, open-source nature, making it an ideal choice for this project’s goal of accessibility and scalability. Unlike other mapping libraries that can be costly or complex, Leaflet.js offers simplicity and powerful real-time map updates without the need for proprietary services. Its straightforward API allowed seamless integration with the tracking system, enabling dynamic visualization of balloon trajectories while keeping the application efficient and user-friendly.

The system incorporates user-friendly features, including customizable data queries and various visualization options. Users can view balloon paths on an interactive map, access raw data, and download the information in multiple formats such as CSV or JSON for analysis. Beyond its application to high-altitude balloons, the platform demonstrates the potential of combining low-power, long-range communication protocols with advanced visualization tools like Leaflet.js. Future enhancements could include altitude visualization, sensor data integration, and applications to drones or wildlife monitoring, making it a versatile solution for real-time tracking and research.
